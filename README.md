# SUNY Topic Modeller

Topic modeling analysis for SUNY (State University of New York) related documents.

## Overview

This project implements topic modeling algorithms to analyze and categorize documents, originally developed for SUNY-related content analysis.

## Features

- Topic extraction from documents
- Document clustering
- Topic visualization

## Technologies

- Java
- Topic Modeling algorithms (LDA)
- Text processing

## Use Cases

- Academic document analysis
- Research paper categorization
- Content discovery
